# Name

Статичная вёрстка

## Установка

```bash
npm i
```

или

```bash
yarn
```

## Запуск

```bash
npm run start
```

или

```bash
yarn start
```

### Собрать файлы без минификации в /build

```bash
npm run build
```

или

```bash
yarn build
```

### Собрать и сжать все файлы для продакшена в /dist

```bash
npm run dist
```

или

```bash
yarn dist
```

## Среда

```bash
node >= 14
```
